# Calculadora Simples em Python

Este é um projeto de uma calculadora simples desenvolvida em Python.

## Funcionalidades

A calculadora oferece as seguintes operações básicas:

*   Soma
*   Subtração
*   Multiplicação
*   Divisão

## Como usar

1.  Certifique-se de ter o Python instalado em sua máquina.
2.  Execute o arquivo `calculadora.py` em seu terminal:

```bash
python calculadora.py
```

3.  O programa exibirá um menu com as opções de cálculo.
4.  Escolha a operação desejada digitando o número correspondente.
5.  Insira os dois números para realizar a operação.
6.  O resultado será exibido na tela.
7.  Para sair da calculadora, escolha a opção "5".

## Estrutura do Código

O código está contido no arquivo `calculadora.py`. Ele consiste em uma função principal `calcular()` que é executada em um loop, permitindo que o usuário realize múltiplos cálculos sem reiniciar o programa.
